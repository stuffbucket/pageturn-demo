An interactive 3D book built with Three.js, TypeScript, and Vite. Features drag-to-turn pages with physics settle, a gravity-based curl shader, rich procedural content, and a Big Buck Bunny video spread.

## Install and run

<Steps>

1. Clone the repository and install dependencies (Node 22 LTS or newer):

   ```bash
   git clone https://github.com/stuffbucket/pageturn-demo.git
   cd pageturn-demo
   npm install
   ```

2. Start the dev server (Vite, default port 5173):

   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) and drag the book pages.

</Steps>

## Controls

- **Click-drag** pages to turn.
- **Arrow keys** or the **prev / next** buttons step a single spread.
- **Right-drag** to orbit; **scroll** to zoom (OrbitControls auto-disable during a page drag).

## Architecture in one paragraph

`src/main.ts` owns the scene, camera, and drag system. `src/book/Book.ts` is the rendering orchestrator — two static spread meshes plus a third animated `turningPageMesh` and a crease shadow strip. `src/book/BookState.ts` is a *pure* state machine (no Three.js imports) that owns the discrete spread index `j` and the turn angle `φ`. Procedural canvas textures live in `src/textures/atlas.ts`. The full breakdown lives in [Architecture &rarr; Overview](/pageturn-demo/architecture/overview/).

:::tip[Where the math lives]
The active page-curl vertex shader is **inlined** in `Book.ts` as the `FLIP_VERT` constant. The legacy files in `src/shaders/` are unused. See [Tilted-crease model](/pageturn-demo/architecture/tilted-crease-model/) for the equations.
:::
